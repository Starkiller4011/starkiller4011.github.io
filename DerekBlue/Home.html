About Me

I am a student of mathematics and logic with a passion for applying both to real-world problems. I have a hand full of fun projects I am currently working on and a couple of finished work that I have done for various groups and organiztions. My most recent accomplishment is the completion of a program for applying a structure function to unevenly sampled time-series data. This program was used for analysis of some of the data in my honors thesis as well as a published paper, the latter of which can be found on my pubblications page. To see the program itself you can visit my Code Projects page which will also contain a link to my GitHub where the program is hosted.
